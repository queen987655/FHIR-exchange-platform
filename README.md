# FHIR Exchange Platform

A healthcare interoperability platform built with Vue 3, Express and HAPI FHIR Server following HL7 FHIR R4 and TW Core IG specifications.

## Technologies

- Vue 3
- Node.js
- Express.js
- HL7 FHIR R4
- TW Core IG
- HAPI FHIR Server
- RESTful API

## Features

- Create Organization Resource
- Create Practitioner Resource
- Create Patient Resource
- Query Patient by ID
- Query Patients by Organization
- FHIR JSON Viewer
- Resource Validation

## Architecture

Vue Frontend
↓
Express Backend
↓
HAPI FHIR Server

## FHIR Resources

- Organization
- Practitioner
- Patient
- Bundle
- OperationOutcome
